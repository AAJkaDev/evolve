### Identity & Visual Language Blueprint

#### 1. Core Metaphor: “Paper for Learning”
Paper signals knowledge acquisition (books, notebooks) and experimentation (doodles, prototypes). Make it tangible with:

-  Black-on-white line-art illustrations (e.g., paper planes, pencils, rockets, torn-edge frames, notebook margins).  
-  Micro-interactions that “scribble” or “erase” on hover.  
-  Light paper-texture background (≤5% contrast) to preserve accessibility.

#### 2. Iconography & Micro-Imagery
Name tags for reusable assets (SVG, 2 px stroke, rounded joins):

| Tag | Usage | Style cue |
| ---- | ---- | ---- |
| paper-plane | progression / quick-start | dashed motion lines |
| folded-corner-page | tool-tips / docs | 45° top-right fold |
| pencil-edit | inline edit | mono-stroke, tilted 18° |
| rocket-launch | AI-powered boost | smoke in three ellipses |
| checklist-pad | task / module | spiral binding top |
| bookmark-ribbon | save / favourite | negative-space cut |
| ruler-plus-compass | analytics / metrics | intersection lines |
| graduation-cap-hover | milestones | tassel animated on hover |
| chat-bubble-scribble | peer discussion | wavy tail |
| light-bulb-sketch | concept / insight | hatched filament |

(Icon grammar derived from Chen’s icon design methodology [1] and education icon libraries [2][3].)

#### 3. Typography

Balanced between “hand-drawn paper” and “serious SaaS”:

| Role | Font | Why |
| ---- | ---- | ---- |
| Display / H1-H2 | “Recoletta” or “Chronicle Display” (Serif) | evokes textbook headings, high legibility at large sizes [4]. |
| Body | “Inter” or “Roboto” (Sans-serif, 1.5 rem line-height) | proven readability on screens [5]. |
| Accent / Quotes | “Kalam” or “Patrick Hand” (Google hand-script) | note-like authenticity without impairing scanning [6]. |

Best practice: max 2 families (serif + sans); add script only for <5% text. Use 400/700 weights; avoid italics for accessibility.

Readable sizes (WCAG 2.2):

– Body 17–18 px  
– Caption 14 px  
– Headings 1.6–2.8 × body scale.

#### 4. Colour Psychology & Dopamine Loop

Although the macro look is black-and-white, strategic colour pops trigger attention and reward circuits [7][8][9]:

Primary Accent (link/buttons)  
-  Dopamine Blue #4285F4 — boosts focus and trust in learning contexts [10]. Contrast 8.6:1 on white.

Secondary Accents  
-  Mentorship Mint #34C9A3 — calm concentration [7].  
-  Achievement Amber #FFB623 — warmth & milestone feedback [9].  
-  Error / Warning Terracotta #E5533C — accessible alternative to pure red.

Dark Mode / Night Study  
Background #121212; Text #E0E0E0; keep accents identical for brand consistency.

Use 60-30-10 rule: 60% white/paper, 30% greys (#F5F5F5 – #9E9E9E for depth), 10% accent.

Animation cue: On task completion flash accent hue +8% saturation for 180 ms to mimic micro-dopamine hit [11].

#### 5. UI & Layout Patterns (SaaS-Edu Benchmarks)

Pattern | Source SaaS (2025) | Fit for Paper Style
--- | --- | ---
Progressive Dashboard → Module Cards | 360Learning, Skillsoft [12] | Cards look like sticky-notes with drop shadows ≤2 dp.
Split “Reader” Layout (content left, linear outline right) | Coursera, ISTE StretchAI [13] | Outline rendered as ruled notebook margin.
Inline AI Coach | Mainstay bot [12] | Speech bubble with hand-drawn tail.
Checklist-Onboarding | ClickUp UX study [14] | Notebook checklist with tick animations.
Global Search + Instant Preview | Modern SaaS pattern library [15] | Search bar inside torn-paper rectangle.

Apply minimalist UI conventions [16][17]: generous whitespace, 8-pt grid, one accent colour per screen, sticky “paper-tab” nav bar.

#### 6. Psychological Hooks

1. **Curiosity** – reveal-on-scroll doodles (Zeigarnik effect keeps users exploring).  
2. **Competence** – visible progress bar styled as pencil filling a ruler.  
3. **Relatedness** – avatar circles drawn as pencil sketches until profile filled.  
4. **Autonomy** – dark/light toggle shown as sun/moon doodle on sticky note.

#### 7. Accessibility & Performance

-  All icons SVG, <24 KB, 2 : 1 stroke-to-canvas ratio.  
-  Colour-blind safe (simulate Deuteranopia; blue/mint unaffected [7]).  
-  Prefer `prefers-reduced-motion` for sketch animations.  
-  Fonts served via `font-display: swap` to prevent FOUT.

✔ Master icon set (24 × 24 & 48 × 48).  
✔ Figma styles: colour tokens + typography scale.  
✔ CSS class map (e.g., `.paper-card`, `.scribble-hover`).  
✔ Brand one-pager for contributors.
